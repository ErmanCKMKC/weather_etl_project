# Weather ETL Project

This project is a simple ETL pipeline built with Python.

It extracts daily weather data from the Open-Meteo public API,
cleans and transforms the data,
loads it into an SQLite database,
and generates basic weather reports.

------

## Technologies

- Python
- Requests (API calls)
- Pandas (Data processing)
- SQLite (Database)
- Git & GitHub

------

## How to Run

1. Install Python
2. Install required libraries:

pip install requests pandas

3. Run:

python main.py

------

## Purpose

This project was developed to practice:

- API data extraction
- Data cleaning
- Data transformation
- Basic ETL pipeline development
- Database loading
